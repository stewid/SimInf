# Detailed summary of a `SimInf_events` object

Shows the number of scheduled events and the number of scheduled events
per event type.

## Usage

``` r
# S4 method for class 'SimInf_events'
summary(object, ...)
```

## Arguments

- object:

  The `SimInf_events` object

- ...:

  Additional arguments affecting the summary produced.

## Value

None (invisible 'NULL').
